CSS specificity ნიშნავს წესების პრიორიტეტულობას, რომელიც განსაზღვრავს რომელი სტილი უნდა ამოქმედდეს, როდესაც რამდენიმე სტილი ეწინააღმდეგება ერთმანეთს. უფრო კონკრეტულად, ეს არის ტექნიკა, რომელიც დაეხმარება ბრაუზერს გაერკვეს, რომელ CSS-დან უნდა შეირჩეს სტილი, როცა სხვადასხვა სელექტორები სავალდებულოდ აირჩევენ ერთსა და იმავე ელემენტს. CSS specificity შეიძლება ამოიცნოთ სელექტორის ტიპით. ვრცელდება შემდეგნაირად: Inline styles (ინლაინ სტილი) — ყველაზე მაღალი პრიორიტეტი. Inline სტილი პირდაპირ HTML ელემენტზეა განსაზღვრული. ID selectors — ძალიან მაღალი პრიორიტეტი. ID-ები უნიკალურია და ერთი გვერდის ფარგლებში მხოლოდ ერთი უნდა იყოს. Class selectors, pseudo-classes, and attributes selectors — უფრო დაბალი პრიორიტეტი, მაგრამ უფრო ზოგადი ვიდრე ID selectors. Element (type) selectors and pseudo-elements — ყველაზე დაბალი პრიორიტეტი. ეს სელექტორები მხოლოდ ელემენტის ტიპს განსაზღვრავენ, არა მის კლასს ან ID-ს. პრიორიტეტების სია: Inline styles (პირველ რიგში) ID selectors (მაგ: #header) Class selectors, pseudo-classes, attribute selectors (მაგ: .class, :hover, [type="text"]) Element (type) selectors, pseudo-elements (მაგ: div, p, ::before)
