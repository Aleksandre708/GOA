CSS (Cascading Style Sheets) არის ენა, რომლის მეშვეობითაც ვებ-გვერდებზე სტილების (სტილების) ფორსიტება ხდება, როგორიცაა ფერები, შრიფტები, განლაგებები და სხვ. იმის აღწერა, თუ როგორ გამოიყენება CSS-ის დეკლარაციები, კუთვნილებები და მნიშვნელობები: CSS დეკლარაცია: ეს არის ცალკეული რეგულაცია, რომელიც განსაზღვრავს, თუ როგორ უნდა იყოს გამოყოფილი გარკვეული HTML ელემენტი. დეკლარაციები შედგება კუთვნილებისგან და მისთვის მითითებული მნიშვნელობისგან. მაგ. css color: blue; აქ, color არის კუთვნილება, ხოლო blue არის მნიშვნელობა. კუთვნილება (Property): ეს არის სპეფიციფიკის წამყვანელი ნაწილებიდან ერთ-ერთი, რომელიც განსაზღვრავს, თუ რის შეცვლა უნდა გაკეთდეს სტილში. მაგალითად, color, font-size, margin. მნიშვნელობა (Value): ეს არის კუთვნილებისთვის განსაზღვრული კონკრეტული პარამეტრების მიცემულობა. ეს შეიძლება იყოს ფერი, ზომა, განლაგება და ა.შ. მაგალითად blue, 16px, 10px. ყოველი მიმდინარე სტრუქტურა CSS ფაილებში შემდეგნაირად შემდგება: css selector { property: value; } მაგალითი: css h1 { font-size: 36px; color: #333333; text-align: center; } ამ მაგალითში: h1 არის სელექტორი, რაც მიუთითებს, რომ სექციაში h1 ელემენტს შევუთავსოთ სტილები. font-size, color, და text-align არის კუთვნილებები. 36px, #333333, და center არის მნიშვნელობები შესაბამისი კუთვნილებებისთვის. ﻿
